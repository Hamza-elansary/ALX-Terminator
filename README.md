# ALX-Terminator
## Background
It’s time to learn some Terminal basics. This game will help you learn how to navigate around a Linux Operating System.
## Objective
Your task is to reach level14 and obtain the final flag.
## Scope
For each level, you will need to move into your new user’s home directory where a set of instructions will be provided for you to reach the next level of the game.
## Task Requirements
- To begin the ALX-Terminator:
    - Open your Kali terminal and type: practical 1 .
    - Connect to the environment using: ssh -p 2222 level1@localhost .
    - When prompted for a password, enter: level1iseasy .
    - Follow the instructions at each level to progress toward level14 and capture the final flag.
    - Upon completion you can exit the remote machine by typing: 
        -  type ~ `then . `- to kill the connection immediately .
        -  type `exit` - until you see your shell return to training-shell.
