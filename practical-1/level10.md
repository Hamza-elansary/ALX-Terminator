### üèÅ The Challenge
In Level 10, you'll find yourself in a situation where the password for Level 11 is nowhere to be found. To complete the game and add your name to the winners' log, you'll need to think outside the box and use your sudo rights to change the password of the Level 11 user.

**the solution method**
- Use the `sudo` command with the `-l` option to get any commands you can type with root privileges.
- Use the `password` command to change the password for level 11.
- When changing your Level 11 password, go to Level 11 and enter "Level 11" using the `su` command and your password.
**Here's what you need to do** :
- In the previous level we learnt how to see what rights our user had.
- Try change the password to one you control. 
- When prompted, enter a new password for the Level 11 user. 
- After successfully changing the password, log in as the Level 11 user
